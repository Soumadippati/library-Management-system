# Library Management System

A simple Library Management System to manage books, users, and transactions such as borrowing and returning books. Useful for schools, colleges, or small libraries.

## Features

- Add, update, and delete books
- Register and manage users/members
- Issue and return books
- Track due dates and overdue returns
- Basic reporting (e.g., list of borrowed books)

## Technologies Used

- Programming Language: [HTML,CSS, PHP]
- Database:  [ MySQL]
- Frontend: [HTML/CSS, JavaScript]
